bootstrap 滚动监听

向监听的元素（通常是 body 位置属性需为relative）
添加 data-spy="scroll" 。

data-target 指定 scroll 后哪里需要对应改变
监听元素上的 id （div#section1）
匹配导航栏上的链接 （a[href="#section1"])
对应触发scroll改变
data-offset 为距顶部多少位置时触发
