使用提示信息，要在jquery中初始化
即设置
$(function(){ $('[data-toggle="tooltip"]').tooltip(); });

使用时， 用data-toggle="tooltip" 来创建
其中 title 设置即为 提示内容
data-placement="top"/"bottom/left/right" 指定出现方位

弹出框类似，点击时出现的提示
$(function(){ $('[data-toggle="popover"]').popover(); });
使用时， 用data-toggle="popover"
title 指定 弹出框标题
data-content 指定 弹出框内容
data-trigger 指定触发状态 可选 focus ， click ， hover
