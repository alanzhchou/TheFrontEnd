学习 bootstrap 的网格系统

1. container 固定长度的容器 修改宽度:最大展示宽带为视口宽度,最小一般不做限制

container 和 container-fluid的区别是
container 在 不同设备下有最大宽度，当设置宽度超过最大宽度时,以最大支持宽度为准

超小设备(<576px):auto
平板(576px < x < 768px): 540px
桌面级(768px < x < 992px): 720px
大桌面(992px < x < 1200px): 992px
超大桌面(>1200px): 1140px -- 电脑浏览器
如上的都为设备逻辑分辨率宽度，
如手机~375px, 平板~768px, 电脑~1080px
