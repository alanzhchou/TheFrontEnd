bootstrap 轮播图

bootstrap 轮播分为三个部分：
1. 位于下方的指示符，
提示是第几副图片为激活状态
2. 轮播图片的主体部分
3. 左右切换的按钮

-------------------
.carousel.slide 标记轮播图容器 data-ride="carousel"

.carousel-indicators 标记指示符列表 
内部用li指向对应的轮播图
        data-target 指定图片
        data-slide-to 指定滑向第几个li

.carousel-inner 标记轮播图主体图片容器 
内部用div.carousel-item 标记图片容器
            其中可放 a>img
            其中 div.carousel-caption 设置图片的描述

a.carousel-control-prev 标记 左右滑动指示icon
href指向 整体容器 data-slide="prev" 或者 "next"
内部包含一个 span.carousel-control-prev-icon
或span.carousel-control-next-icon
